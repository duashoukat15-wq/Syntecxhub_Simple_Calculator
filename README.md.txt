# Simple Calculator (Command Line)

This project is a simple command-line calculator built using Python as part of the Syntecxhub internship task.

## Features
. Supports addition (+), subtraction (-), multiplication (*), and division (/)
.Handles divide-by-zero errors
. Validates user input
.Menu-based system for repeated calculations
. Clear option
. Modular code with separate functions for easy testing

## Technologies Used
.Python 3

## How to Run the Project
1. Install Python 3 on your system
2. Open terminal in the project directory
3. Run the following command:

   python calculator.py

## Project Structure
. calculator.py : Main calculator program
. README.md : Project documentation
